# Quantitative Portfolio Optimizer

A professional-grade portfolio optimization tool implementing Modern Portfolio Theory and advanced quantitative techniques.

## Features

- **Modern Portfolio Theory**: Mean-variance optimization (Markowitz Model)
- **Multiple Objectives**: Maximize Sharpe ratio, minimize volatility, maximize return
- **Advanced Models**: Black-Litterman model with investor views
- **Risk Analysis**: VaR, CVaR, diversification ratios, maximum drawdown
- **Professional Visualization**: Interactive efficient frontier and allocation charts
- **Flexible Constraints**: No shorting, maximum weight limits

## Usage

### Web Dashboard
```
streamlit run dashboard.py
```
# or
```
python main.py dashboard
```
